Blockchain-Enabled Consent Management for Secure Healthcare Data Access is a decentralized application that allows patients to control and manage access to their medical data using blockchain technology. Patients can grant, limit, or revoke data access to verified doctors through smart contracts, ensuring transparency, security, and auditability.
The system stores only cryptographic hashes of medical data on-chain to preserve privacy while maintaining data integrity. All consent actions such as access requests, approvals, expirations, and revocations are recorded immutably on the blockchain. A Web3-based frontend enables interaction via MetaMask without relying on a centralized backend.
